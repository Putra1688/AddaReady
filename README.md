Sebuah aplikasi web modern berbasis **Next.js 16** yang dirancang sebagai fondasi digital utama untuk ekosistem **Sumber Baku Jaya**. Project ini mengedepankan performa tinggi, skalabilitas, dan pengalaman pengguna yang optimal.

## 🚀 Keunggulan Teknis (Project Value)

*   **⚡ Next-Gen Performance**: Menggunakan **Next.js 16** (App Router) untuk rendering sisi server yang sangat cepat dan optimasi aset otomatis.
*   **⚛️ React 19 Architecture**: Implementasi terbaru dari engine React untuk manajemen state dan transisi UI yang lebih halus.
*   **🎨 Tailwind CSS v4**: Styling masa depan dengan compiler high-performance yang memastikan ukuran CSS tetap minimal dan fleksibel.
*   **🛡️ Strict Type Safety**: Seluruh codebase dibangun dengan **TypeScript** untuk meminimalisir error dan mempermudah pemeliharaan jangka panjang.
*   **📂 Modular Structure**: Arsitektur folder yang rapi dan terukur, siap untuk pengembangan fitur kompleks di masa mendatang.

## 🛠️ Tech Stack

| Layer | Teknologi |
| :--- | :--- |
| **Framework** | [Next.js 16.2 (App Router)](https://nextjs.org/) |
| **Library UI** | [React 19](https://react.dev/) |
| **Styling** | [Tailwind CSS v4](https://tailwindcss.com/) |
| **Language** | [TypeScript 5](https://www.typescriptlang.org/) |
| **Font** | [Geist Sans & Mono](https://vercel.com/font) |
| **Linter** | [ESLint 9](https://eslint.org/) |

## 📁 Struktur Inti Proyek

```bash
├── app/              # Routing, Page, dan Layout (Next.js App Router)
├── public/           # Aset statis (Gambar, Ikon, SVG)
├── next.config.ts    # Konfigurasi spesifik framework Next.js
├── tsconfig.json     # Aturan ketat pengembangan TypeScript
└── package.json      # Dependensi dan script project
```
